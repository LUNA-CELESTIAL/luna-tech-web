# 📜 LUNA's Constellations - Official Web App

LUNA's Constellations（うさぎのお星さまファクトリー）の公式Webサイト（`luna-stella.jp`）のフロントエンドリポジトリです。

---

## 🛠️ 技術スタック & 動作環境

- **Framework:** Next.js (App Router) / React
- **Language:** TypeScript / Node.js
- **Deployment:** Vercel (GitHub CI/CD 自動連携)
- **Target OS/Arch:** Ubuntu 24.04 LTS (aarch64 / ARM64)
